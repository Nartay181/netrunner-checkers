# ⚡️ Netrunner Checkers — Cyberpunk Board Platform

![Netrunner Checkers Main HUD](/public/assets/main.png)

### 📝 О проекте (Elevator Pitch)
* **Что я сделал:** Я переосмыслил классическую настольную игру и создал высокотехнологичную веб-платформу для игры в шашки в сеттинге киберпанка. Это не просто игровое поле, а полноценная экосистема с real-time мультиплеером, умным ИИ-тренером, динамическим расчетом рейтинга Elo через Supabase и продвинутым трехколоночным HUD-интерфейсом.
* **Для кого это:** Проект разработан для геймеров и любителей настольных стратегий, которые ценят глубокую атмосферу и соревновательный дух, а также для менторов и комиссии **nFactorial School (Stage 2)** как демонстрация бескомпромиссного уровня веб-разработки.
* **Почему это ценно:** Я ушел от скучных плохих стандартов веб-игр. Ценность платформы в синергии визуала и сложных технологий: здесь работает честный бэкенд без фейковых данных, динамическая фильтрация игроков по региональным узлам (например, Алматы), безопасная авторизация через Google OAuth и интерактивный ИИ, который помогает игроку совершенствовать тактику в реальном времени.

## 🚀 Функционал платформы и Архитектура (Key Features)

### 1. Symmetrical 3-Column HUD Interface `[ РЕАЛИЗОВАНО ]`
* **Идеальный визуальный баланс:** Интерфейс десктопной версии переработан в профессиональную трехколоночную сетку, исключающую пустые зоны.
* **Левое крыло:** Панель профиля оператора (`Player Profile`) и история личных матчей (`Game History`).
* **Центр:** Интерактивное неоновое игровое поле для шашек с полной игровой логикой.
* **Правое крыло:** Живой лог системных событий (`Terminal Output`) и глобальная таблица лидеров (`Netrunner Leaderboard`).

### 2. Secure Identity Layer & Google OAuth `[ РЕАЛИЗОВАНО ]`
![Netrunner Checkers Google](/public/assets/google.png)
* **Бесшовная авторизация:** Быстрый вход в один клик через Google Auth или зашифрованный терминал регистрации.
* **Автоматический парсинг метаданных:** Система вытягивает реальное имя пользователя из Google-аккаунта при первой авторизации, предотвращая появление дубликатов и фейковых ников.
![Netrunner Checkers Main HUD](/public/assets/meta.png)
* **Bypass Email Validation:** Настройки бэкенда оптимизированы для мгновенного создания аккаунтов без ожидания писем-подтверждений.

### 3. Динамический Leaderboard & Elo Rating Matrix `[ РЕАЛИЗОВАНО ]`
![Netrunner Checkers Main HUD](/public/assets/leaderboard.png)
* **Интеграция с PostgreSQL:** Рейтинги подтягиваются в реальном времени из таблицы `public.profiles`.
* **Система Elo:** Динамический расчет уровня мастерства (все новые аккаунты по умолчанию получают `1000 ELO`и статус `BRONZE RUNNER`).
* **Локальный фильтр (Almaty Node):** Быстрое переключение между глобальной сетью (`[ ALL NETWORKS ]`) и локальным узлом (`[ ALMATY NODE ]`) для фильтрации игроков из Алматы.

### 4. Архивация & Game History Archive `[ РЕАЛИЗОВАНО ]`
![Netrunner Checkers Main HUD](/public/assets/gamehistory.png)
* **Выделенная таблица данных:** Все завершенные сессии логируются в изолированную таблицу `public.match_history` (разведение real-time комнат и архивных данных).
* **Журнал транзакций:** Панель отображает историю игр текущего пользователя в формате: `[DATE] | [OPPONENT] | [RESULT] | [ELO CHANGE]`.

### 5. AI Cyber-Coach v2.0 & Полный Адаптив `[ РЕАЛИЗОВАНО ]`
* **Тактический оверлей:** ИИ-подсказки разворачиваются в эффектный полноэкранный HUD с typewriter-эффектом (имитация хакерского взлома и печатающегося текста).
* **Responsive Layout:** Интерфейс плавно адаптируется под экраны планшетов и мобильных устройств, перестраивая трехколоночную сетку в удобный вертикальный стек без горизонтального скролла (`overflow-x-hidden`).

## 🛠 Технологический Стек (Tech Stack)

| Компонент | Технология | Назначение |
| :--- | :--- | :--- |
| **Frontend Framework** | React / Next.js | Архитектура веб-платформы, быстрый клиентский и SSR роутинг |
| **Backend & Auth** | Supabase (PostgreSQL) | Сессии, Google OAuth, хранение профилей и триггеры базы данных |
| **Styling & HUD UI** | Tailwind CSS | Тематический sci-fi дизайн, неоновое свечение, кастомные сетки |
| **Typechecking** | TypeScript | Строгая типизация кодовой базы для предотвращения рантайм-ошибок |
| **Deployment** | Vercel | Высокопроизводительный хостинг фронтенда с автоматическим CI/CD |

## 🛠 Как запустить проект локально

```bash
# 1. Клонировать репозиторий проекта
git clone [https://github.com/your-username/netrunner-checkers.git](https://github.com/your-username/netrunner-checkers.git)

# 2. Перейти в директорию и установить все зависимости
cd netrunner-checkers
npm install

# 3. Настроить переменные окружения (.env.local)
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key

# 4. Запустить локальный сервер для разработки
npm run dev
```
